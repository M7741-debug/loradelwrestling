README - L'ORA DEL WRESTLING (Progetto Sito Web Statico)

Scopo del sito:
---------------
Questo sito nasce come supporto gratuito, leggero e indipendente ai podcast e ai contenuti YouTube di "L'ora del Wrestling".
Serve per raccogliere articoli, dare informazioni ai fan, rispondere a domande e mantenere un contatto diretto con chi segue il progetto.

NON è un sito con login, forum, news, recensioni o gestione staff.
Non è una testata giornalistica né una piattaforma di discussione.
Lo scopo è chiaro: supportare e valorizzare ciò che già esiste.

Struttura del sito:
-------------------
1. index.html         - Homepage con presentazione, contatti, link podcast/social
2. articoli.html      - Elenco articoli e link ai singoli pezzi
3. articolo_template.html - Modello per scrivere nuovi articoli facilmente
4. disclaimer_privacy.html - Disclaimer legale + Privacy
5. (spazio per logo e immagini) - gestito manualmente
6. (eventuali futuri .html) - es. "chi_siamo.html", "domande.html"

Piattaforma consigliata:
-------------------------
- Hosting gratuito su Netlify (HTTPS attivo, stabile, senza pubblicità)
- Modifiche tramite Blocco Note o editor HTML semplice

Email:
------
Proteggere l’indirizzo email dallo spam è fondamentale.
È stato inserito uno script JavaScript per mostrarla in modo sicuro.

Note finali:
------------
- Nessun tracciamento, cookie o raccolta dati
- Nessuna password o login richiesto
- Ogni aggiornamento va fatto manualmente (copiando/modificando i file)

Questo progetto è fatto con semplicità, passione e spirito indipendente.
